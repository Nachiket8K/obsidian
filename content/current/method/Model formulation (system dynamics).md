# Model formulation (system dynamics)

## Upward links
- [[The Delft Method for System Dynamics - Source]]

## Summary
The SD method phase that translates a conceptual model into equations, stocks, flows, auxiliaries, and numerical settings. It emphasises preserving meaning, dimensional consistency, and dynamic realism while building a runnable model.

## Downward links
- [[Delft SD Ch 03 - Model formulation 1]]
- [[Delft SD Ch 04 - Model formulation 2]]
- [[Delft SD Ch 07 - Formulation]]

## Core ideas
- Turn conceptual structure into explicit stock-flow equations.
- Choose numerical methods and settings that fit the model logic.
- Use standard structures where they clarify dynamic mechanisms.
- Keep the formulation tied to the original problem and boundaries.
