# Dynamic Adaptive Planning (DAP)

## Upward links
- [[DMDU - Source]]

## Summary
A planning approach that combines immediate actions with contingent actions so plans can evolve as the world changes. DAP makes vulnerabilities, opportunities, signposts, and triggers explicit from the start.

## Downward links
- [[DMDU Ch 03 - Dynamic Adaptive Planning (DAP)]]
- [[DMDU Ch 08 - Dynamic Adaptive Planning (DAP) - The Case of Intelligent Speed Adaptation]]

## Core ideas
- Begin with an initial plan, then improve it with adaptive actions.
- Track signposts and triggers that indicate when adjustment is needed.
- Treat learning as part of the plan rather than an afterthought.
- Make contingent responses explicit before implementation.
