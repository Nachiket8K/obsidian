# Causal map

## Upward links
- [[Policy Analysis of Multi-Actor Systems - Source]]

## Summary
A causal map represents how variables or factors influence one another so analysts can reason about interactions, feedbacks, and possible leverage points.

## Downward links
- [[PAMS Ch 03 - Systems Analysis]]
- [[PAMS Ch 03 S04 - Mapping Causal Relations]]

## Core ideas
- Represent influence structure before formal quantification.
- Help identify reinforcing, balancing, or indirect effects.
- Support discussion of system behaviour and intervention logic.
