# Model evaluation (system dynamics)

## Upward links
- [[The Delft Method for System Dynamics - Source]]
- [[EMA Workbench Documentation - Source]]

## Summary
The SD phase that checks whether a model is coded correctly, fit for purpose, interpretable, and useful across uncertainty. In this book it includes verification, validation, interpretation of results, and scenario development.

## Downward links
- [[Delft SD Ch 05 - Model evaluation]]
- [[Delft SD Ch 08 - Evaluation]]

## Core ideas
- Separate internal correctness from fit-for-purpose credibility.
- Use structural and behavioural tests rather than relying on one metric.
- Interpret model behaviour by linking it back to structure.
- Use scenario development to examine robustness under uncertainty.
