# Objectives tree

## Upward links

## Summary
An objectives tree organizes goals and subgoals hierarchically so analysts can make evaluation criteria and strategic intent more explicit.

## Downward links
- [[PAMS Ch 03 - Systems Analysis]]

## Core ideas
- Break broad objectives into smaller, related subobjectives.
- Reveal trade-offs and missing criteria.
- Support communication about what success means.
