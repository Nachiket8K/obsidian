# Dynamic Adaptive Policy Pathways (DAPP)

## Upward links
- [[DMDU - Source]]

## Summary
An adaptive planning method that organizes decisions as pathways through time, with adaptation tipping points defining when a switch is needed. DAPP is especially useful when timing and sequencing matter as much as initial choice.

## Downward links
- [[DMDU Ch 04 - Dynamic Adaptive Policy Pathways (DAPP)]]
- [[DMDU Ch 09 - Dynamic Adaptive Policy Pathways (DAPP) - From Theory to Practice]]

## Core ideas
- Map multiple pathways instead of committing to one permanent sequence.
- Use adaptation tipping points to know when a path no longer works.
- Sequence actions to preserve flexibility.
- Link long-term pathways to near-term decisions and monitoring.
