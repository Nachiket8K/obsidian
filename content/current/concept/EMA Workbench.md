# EMA Workbench

## Upward links
- [[EMA Workbench Documentation - Source]]

## Summary
EMA Workbench is a Python framework for exploratory modeling, computational experimentation, and decision-support analysis under uncertainty. In this vault it acts as a practical bridge between DMDU methods, System Dynamics models, agent-based models, and simulation result analysis.

## Key points
- It provides a shared experimentation layer for models built in multiple environments.
- It combines uncertainty sampling, policy search, robustness-oriented analysis, and visualization in one workflow.
- It is especially relevant where DMDU ideas need executable support rather than remaining only conceptual.
