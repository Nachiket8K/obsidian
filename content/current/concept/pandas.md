## Summary
pandas is the core Python library for tabular data structures and data analysis workflows. In this vault it acts as the foundational non-spatial data layer beneath GeoPandas and many simulation, modeling, and data-processing workflows.

## Downward links
- [[Python Library Documentation/pandas/Home|pandas home]]

## Key points
- pandas provides high-performance data structures and analysis tools for Python.
- It is foundational to many data preparation, transformation, and analysis workflows.
- GeoPandas extends pandas-style structures into the geospatial domain.
