# Mesa framework

## Upward links
- [[Mesa Documentation - Source]]

## Summary
Mesa is a Python framework for building, analyzing, and visualizing agent-based models. It provides a practical implementation layer that can support exploratory simulation, policy experiments, and interactive model communication.

## Key points
- Mesa separates modeling, analysis, and visualization modules while keeping them interoperable.
- It supports both step-based activation and event-based scheduling.
- It is well suited for ABM workflows that need parameter exploration, data collection, and interactive inspection.
