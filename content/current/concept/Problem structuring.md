# Problem structuring

## Upward links
- [[Policy Analysis of Multi-Actor Systems - Source]]

## Summary
Problem structuring is the early-stage activity of clarifying what the problem is, whose problem it is, how it is framed, and which lines of analysis are worth pursuing.

## Downward links
- [[PAMS Ch 01 - Introduction]]
- [[PAMS Ch 02 - Problem Formulation in Complex Environments]]

## Key points
- It precedes detailed solution search in ambiguous environments.
- It makes hidden assumptions, values, and boundaries explicit.
- It helps separate competing diagnoses that would otherwise be mixed together.
