# Primary and secondary rules

## Upward links
- [[DMDU - Source]]

## Summary
Planned Adaptation distinguishes substantive rules that govern a system from the meta-rules used to recognize, revise, and adjudicate those substantive rules. This distinction helps analyse adaptive governance capacity.

## Downward links
- [[DMDU Ch 13 - A Conceptual Model of Planned Adaptation (PA)]]

## Key points
- Primary rules shape behaviour in the domain itself.
- Secondary rules determine how adaptation can occur.
- The distinction explains why some systems can learn and revise while others cannot.
