# AgentSet

## Upward links
- [[Mesa Documentation - Source]]

## Summary
`AgentSet` is Mesa's main structure for storing, querying, grouping, and activating agents. It is central to how Mesa turns a collection of agents into an operational simulation model.

## Downward links
- [[Python Library Documentation/Mesa/overview|Mesa overview]]

## Key points
- `AgentSet` supports selection, grouping, aggregation, and activation operations.
- Activation order and grouping choices can materially affect model outcomes.
- It serves as a practical handle for turning model structure into experiments.
