# Systems analysis

## Upward links
- [[Policy Analysis of Multi-Actor Systems - Source]]

## Summary
Systems analysis studies how problem elements, means, ends, causes, and boundaries relate so analysts can reason about consequences, leverage points, and intervention opportunities.

## Downward links
- [[PAMS Ch 03 - Systems Analysis]]

## Key points
- It clarifies how parts of a problem system influence one another.
- It helps move from vague concerns to more structured representations.
- It is especially useful before building formal or simulation models.
