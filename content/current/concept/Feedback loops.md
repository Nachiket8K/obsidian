# Feedback loops

## Upward links
- [[The Delft Method for System Dynamics - Source]]

## Summary
Feedback loops are chains of causality that close back on themselves and generate endogenous behaviour. In System Dynamics they are used to explain reinforcing growth, balancing adjustment, oscillation, and other dynamic patterns.

## Downward links
- [[Delft SD Ch 01 - Introduction to System Dynamics]]
- [[Delft SD Ch 02 - The first modelling cycle]]
- [[Delft SD Ch 06 - Problem Formulation and Conceptualisation]]

## Key points
- Reinforcing loops amplify change and balancing loops oppose it.
- Delays inside feedback loops can make dynamics harder to interpret or control.
- Feedback reasoning discourages symptom-only interventions.
