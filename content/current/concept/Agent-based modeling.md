## Summary
Agent-based modeling (ABM) represents systems as interacting autonomous entities whose local behaviour generates system-level dynamics. In this vault, ABM acts as a bridge between Mesa implementation details and broader DMDU/System Dynamics questions about exploration, emergence, policy design, and complex systems.

## Key points
- ABM is useful when heterogeneous actors, local rules, and interaction structure matter.
- ABM complements System Dynamics by emphasizing micro-level behavior and emergence rather than aggregate stock-flow structure alone.
- In a DMDU context, ABM can be used as an exploratory model class for testing policies across many plausible behavioural settings.
