# Decision-support framework

## Upward links
- [[DMDU - Source]]
- [[Policy Analysis of Multi-Actor Systems - Source]]

## Summary
The introductory DMDU framework connects external forces, policies, system relationships, outcomes, and stakeholder values. It is a reusable lens for structuring long-term policy problems before selecting a method.

## Key points
- Helps separate controllable levers from external forces.
- Makes stakeholder values explicit instead of burying them inside model assumptions.
- Provides a common frame across methods such as RDM, DAP, DAPP, and DS.
