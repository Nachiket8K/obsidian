# Scenario development

## Upward links
- [[The Delft Method for System Dynamics - Source]]
- [[Policy Analysis of Multi-Actor Systems - Source]]

## Summary
Scenario development uses SD models to examine how behaviour changes across different plausible futures or input combinations. It is a core bridge concept from System Dynamics into DMDU-style robustness analysis.

## Downward links
- [[Delft SD Ch 05 - Model evaluation]]
- [[Delft SD Ch 08 - Evaluation]]
- [[PAMS Ch 05 - Exploring the Future]]

## Key points
- Broadens the range of plausible behavioural futures explored with the model.
- Supports testing policies beyond a single base case.
- Provides a natural bridge to robustness and scenario discovery.
