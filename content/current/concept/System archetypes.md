# System archetypes

## Upward links
- [[The Delft Method for System Dynamics - Source]]

## Summary
System archetypes are recurring feedback structures that generate familiar classes of behaviour. They provide reusable qualitative explanations and can guide intervention design.

## Downward links
- [[Delft SD Ch 04 - Model formulation 2]]
- [[Delft SD Ch 06 - Problem Formulation and Conceptualisation]]

## Key points
- Capture common structural causes of recurring dynamic patterns.
- Support early qualitative diagnosis before full quantification.
- Help relate structure to intervention opportunities and unintended effects.
