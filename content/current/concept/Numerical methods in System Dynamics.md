# Numerical methods in System Dynamics

## Upward links
- [[The Delft Method for System Dynamics - Source]]

## Summary
Numerical methods in SD determine how continuous stock-flow equations are integrated through time. The choice of method and time step affects accuracy, stability, and how faithfully discontinuities are represented.

## Downward links
- [[Delft SD Ch 07 - Formulation]]

## Key points
- Method choice should match model structure and behaviour.
- Time step settings can create artefacts if they are poorly chosen.
- Technical formulation choices still shape substantive conclusions.
