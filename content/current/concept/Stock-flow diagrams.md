# Stock-flow diagrams

## Upward links
- [[The Delft Method for System Dynamics - Source]]

## Summary
Stock-Flow Diagrams are the main quantitative representation used to construct SD models. They make accumulations, rates, and supporting variables explicit in a form that can be simulated.

## Downward links
- [[Delft SD Ch 02 - The first modelling cycle]]
- [[Delft SD Ch 03 - Model formulation 1]]
- [[Delft SD Ch 07 - Formulation]]

## Key points
- Translate conceptual models into a simulation-ready structure.
- Clarify which variables accumulate and which rates change them.
- Support communication about model architecture and assumptions.
