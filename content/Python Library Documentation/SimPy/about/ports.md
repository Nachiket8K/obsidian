# Ports and comparable libraries

Re-implementations of SimPy and libraries similar to SimPy are available in the
following languages:

- C#: [SimSharp](https://github.com/abeham/SimSharp) (written by Andreas Beham)
- Julia: [ConcurrentSim](https://github.com/JuliaDynamics/ConcurrentSim.jl)
- R: [Simmer](https://github.com/r-simmer/simmer)
